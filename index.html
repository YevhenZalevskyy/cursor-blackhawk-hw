Git cheat sheet .

ШПАРГАЛКА

git config –global user.name blabla - ввести ім’я користувача

git config –global user.email blabla@gmail.com - ввести email користувача

git init - створити репозиторій

git status – подивитися стан репозиторія, які файли були змінені чи з’явилися нові

git add – додати файл у stage area для наступного коміта

git commit — зробити коміт (внести змінення у зліпок проекту)

git log - подивитися історію комітів, знайти потрібний

git checkout – перейти до певного коміту

git branch - перейти до іншої гілки

git push - відправити змінення з локального репозиторія до віддаленого

git pull - зтягнути змінення з віддаленого репозиторія до локального

git merge - об'єднати коміти з різних гілок (branches)

git clone - клонувати репозиторій


gitignor файл це файл, якій нам не потрібно вносити в копію (зліпок) проекту

git revert - відкотити стан репозиторію до певного коміту

git reset - скидання незакомічених змінень до стану останнього коміту
